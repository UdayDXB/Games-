# 🐉 Toothless: The Night Fury Flight
**Mission:** Rescue the Blue Dragons and avoid the Dragon Hunters!

## 🕹️ How to Play
1. **Download** this repository as a ZIP or clone it.
2. Make sure you have **Python** installed.
3. Open your terminal and type: `pip install pygame`
4. Run `Toothless_The_Night_Fury_Flight.py` to begin the mission!

## 🚀 Game Features
- **XP System**: Earn 150 XP for every dragon rescued.
- **Adaptive Difficulty**: A new Hunter joins the sky every 5 rescues!
- **Mission Briefing**: Read the instructions before you take flight.
- **Quit Button**: Use the "Leave Game" button to exit safely.

## 🛠️ Built With
- **Python**
- **Pygame Library**
