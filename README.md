# 🎮 The Elite Python Game Collection
Welcome to my official coding repository! This project features two fully functional games built from scratch using Python.

## 📁 Explore My Projects

### 🐉 [Toothless: The Night Fury Flight](./Toothless_The_Night_Fury_Rescue/)
A high-intensity rescue mission built with **Pygame**.
*   **Mission:** Rescue blue dragons while dodging dragon hunters.
*   **Key Feature:** Dynamic Difficulty—more hunters appear every 5 rescues!
*   **Graphics:** Custom PNG sprites for a retro gaming feel.

### 🪨 [Rock Paper Scissors](./Rock-Paper-Scissors/)
A classic battle of logic and luck.
*   **Gameplay:** Test your strategy against the computer's random moves.
*   **Structure:** Clean Python logic using the `random` module.

---

## 🛠️ Global Requirements
To play these games, you will need:
1. **Python 3.x** installed on your computer.
2. The **Pygame** library (for the Dragon game). 
   - Install it by typing `pip install pygame` in your terminal.

## 🚀 How to Run
1. Download this repository as a **ZIP**.
2. Extract the folder.
3. Open the folder of the game you want to play in **VS Code**.
4. Run the `.py` file and enjoy!
