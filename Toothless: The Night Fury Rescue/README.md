# 🐉 Toothless: The Night Fury Flight
**Mission:** The Dragon Hunters have captured the hatchlings! You must fly Toothless and bring them home safely.

## 🕹️ Controls
- **Arrow Keys:** Fly the Night Fury
- **Spacebar / Click:** Start the Mission
- **Red Button:** Leave Game

## 🚀 Pro Features
- **XP Engine:** Earn 150 XP for every rescue.
- **Difficulty Scaling:** A new Hunter appears every 5 rescues!
- **Mission Briefing:** Instructions appear before every flight.

## 🛠️ Requirements
- **Python 3.x**
- **Pygame Library** (`pip install pygame`)
